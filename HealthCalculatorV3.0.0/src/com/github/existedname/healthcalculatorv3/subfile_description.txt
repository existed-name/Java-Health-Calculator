(一)重构前
1. app 包: 存放 MainApplication 类, 也就是主程序

2. model 包: 存放实体类, 用于创建对象/对象数组, 储存相关数据
  2.1 UserBodyInfo 类: 存放各种身体数据
  2.2 User 类: 继承自 UserBodyInfo 类, 存放用户基本信息
  2.3 HealthMetricIntroduction 类: 创建对象数组 introductionArr, 储存各种身体指标的介绍信息
  2.4 BasicBodyParameter 类: 创建对象数组 idealRangeArr, 储存各种身体指标的理想范围
  2.5 HealthMetricIdealRange 类: 储存各种身体指标在不同数值范围下对应的身体情况( 用于给出健康分析 )

3. service 包: 存放 Service 类
    调用 util 包实现计算、分析, 自身具有打印前言 & 功能清单、介绍、给出理想范围等功能

4. util 包: 存放用于计算、分析的工具类
  4.1 BMICalculator, WHRCalculator, BFRCalculator, BRICalculator, BMRCalculator, BSACalculator 类: 都是顾名思义, 计算并返回某个值
  4.2 TDEECalculator 类: 除了计算, 还有打印活动系数表
  4.3 HealthMetricAnalyzer 类: 调用 BasicBodyParameter 类, 作用是分析身体指标


(二)重构后
1. app 包: 存放 MainApplication 类( 主程序 )
    startApp 方法作为核心方法: 首先展示前言、功能编号, 再进入游戏循环, 检查用户输入的编号并跳转到对应功能


2. model 包: 存放实体类, 用于创建对象/对象数组, 储存相关数据
  2.1 UserBodyInfo 类: 存放各种身体数据
  2.2 User 类: 继承自 UserBodyInfo 类, 存放用户基本信息
  2.3 HealthMetricIntroduction 类: 创建对象数组 introductionArr, 储存各种身体指标的介绍信息
  2.4 BasicBodyParameter 类: 创建对象数组 idealRangeArr, 储存各种身体指标的理想范围
  2.5 HealthMetricIdealRange 类: 储存各种身体指标在不同数值范围下对应的身体情况( 用于给出健康分析 )

3. service 包:

  3.1 CalculationService 计算各种身体指数
  3.2 AnalysisService 分析各种身体指数
  3.3 IntroductionService 讲解...
  3.4 IdealRangeService 给出...的理想范围/理想值
  3.5 ComparisonService 对比某些身体指数的不同公式计算结果

存放 Service 类
  调用 util 包实现计算、分析, 自身具有打印前言 & 功能清单、介绍、给出理想范围等功能

  4.1 calculator 包
    4.1.1 BMICalculator, WHRCalculator, BFRCalculator, BRICalculator, BMRCalculator, BSACalculator 类: 都是顾名思义, 计算并返回某个值

  4.3 convertor 包

  4.4 inputter 包
